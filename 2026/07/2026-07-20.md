## [1/10] PagedWeight: Efficient MoE LLM Serving with Dynamic Quality-Aware Weight Quantization

- **저자**: Yuchen Yang, Yifan Zhao, Anisha Dasgupta
- **공개일**: 2026-07-17
- **링크**: https://arxiv.org/abs/2607.16184v1

**요약**

MoE LLM serving 중 가중치와 KV-cache 간 memory 경쟁을 해결하기 위해, 런타임에서 전문가별 정밀도를 동적으로 최적화하는 PagedWeight Dynamic Weight Quantization 기법을 제안합니다.
정확도, memory, throughput 간 tradeoff를 자동 조절하여 FP16 수준의 quality를 유지하며 GPU memory를 최대 72% 절감하고 1.94배 이상 처리량을 향상시킵니다.
기존 quantization 방법 대비 동일 memory budget 내에서 quality를 39.3%까지 개선하며, 고효율 LLM serving 시스템 구축에 실질적인 기여를 합니다.
