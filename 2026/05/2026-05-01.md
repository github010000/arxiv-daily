## [3/10] Select to Think: Unlocking SLM Potential with Local Sufficiency

- **저자**: Wenxuan Ye, Yangyang Zhang, Xueli An
- **공개일**: 2026-04-29
- **링크**: https://arxiv.org/abs/2604.26940v1

**요약**

SLM과 LLM 간 추론 격차를 해소하기 위해 LLM을 생성기가 아닌 SLM의 Top-K 후보 선택기로 재구성한 **SELECT TO THINK (S2T)** 및 **S2T-LOCAL** 방법론을 제안합니다.
**Local Sufficiency** 가설을 바탕으로 LLM의 선택 로직을 SLM에 **Distillation**하여, 추론 시 외부 LLM 의존 없이 자체적으로 후보를 재순위화(Autonomous Re-ranking)할 수 있도록 합니다.
이를 통해 1.5B SLM이 32B LLM의 선택을 95% 정확도로 대체하며, 단일 경로 추론으로 8-path self-consistency 수준의 성능(평균 24.1% 향상)을 달성해 효율적인 배포를 실현합니다.

## [6/10] ClassEval-Pro: A Cross-Domain Benchmark for Class-Level Code Generation

- **저자**: Yeheng Chen, Chaoxiang Xie, Yuling Shi
- **공개일**: 2026-04-29
- **링크**: https://arxiv.org/abs/2604.26923v1

**요약**

ClassEval-Pro는 자동화 파이프라인과 LLM Judge Ensemble을 통해 11개 도메인의 300개 Class-Level Code Generation benchmark를 구성한 평가 프레임워크입니다.
5개 frontier LLM과 5개 generation strategy를 평가하여 bottom-up 접근이 약한 모델 성능을 9.4%p 향상시킴을 입증했습니다.
데이터 오염에 강인한 확장 가능한 benchmark를 제공하며, logic/dependency 오류 분석을 통해 Class-Level 생성의 핵심 bottleneck을 제시했습니다.
