## [7/10] Strait: Perceiving Priority and Interference in ML Inference Serving

- **저자**: Haidong Zhao, Nikolaos Georgantas
- **공개일**: 2026-04-30
- **링크**: https://arxiv.org/abs/2604.28175v1

**요약**

**Strait**은 GPU concurrent execution 시 data transfer contention과 kernel interference를 반영한 adaptive prediction model로 latency를 추정하고, priority-aware scheduling으로 traffic을 차별 처리하는 serving system입니다.
고부하 워크로드에서 high-priority task의 deadline violation을 1.02~11.18%p 감소시키며, low-priority task에는 허용 가능한 성능 overhead만 발생시킵니다.
software-defined preemption 대비 더 equitable한 자원 분배를 제공해, on-premises ML inference serving의 SLA 준수와 운영 효율성을 실용적으로 향상시킵니다.

- **링크**: https://arxiv.org/abs/2604.28167v1

**요약**
