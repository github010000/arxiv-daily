## [3/10] AdaCodec: A Predictive Visual Code for Video MLLMs

- **저자**: Haowen Hou, Zhen Huang, Zheming Liang
- **공개일**: 2026-06-01
- **링크**: https://arxiv.org/abs/2606.02569v1

**요약**

기존 Video MLLM의 프레임별 독립 인코딩 한계를 극복하기 위해, 프레임 간 예측 오차에 따라 시각적 토큰 할당을 동적으로 조정하는 predictive visual code 기반 AdaCodec을 제안합니다.
예측 비용이 높을 때는 전체 reference frame 토큰을 사용하고, 나머지는 motion 및 prediction residual을 P-tokens으로 압축 인코딩하여 시각적 중복을 제거합니다.
시각적 토큰 예산을 1/7로 줄이면서도 벤치마크 성능을 전반적으로 향상시키고, time-to-first-token을 9.26초에서 1.62초로 단축하여 실제 Video MLLM 시스템의 효율성과 실시간 응답성을 크게 개선합니다.
